# Image Assets

Please upload your images to this folder.

## Required Images for the App:
1. `school-logo.png` - The primary school logo (blue circular logo).
2. `alumni-logo.png` - The alumni association logo (green circular logo).

## How to upload:
1. Open the file explorer in the code editor.
2. Navigate to `public/assets/images/`.
3. Right-click and select "Upload" or drag and drop your files here.
4. Ensure the filenames match exactly as listed above.
